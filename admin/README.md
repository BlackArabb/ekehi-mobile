## Getting Started

### Prerequisites

- Node.js 18+
- pnpm package manager

### Installation

```bash
cd admin
pnpm install
```

### Development

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Building for Production

```bash
pnpm build
pnpm start
```
